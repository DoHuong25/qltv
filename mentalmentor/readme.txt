Files Description:
1) mentalmentor.exe - main executable file
2) uninstall.exe - uninstaller 
3) mentalmentor_crashpad_handler.exe - crash handler  
4) QtWebEngineProcess.exe - Qt web engine 
5) other files - Qt libraries necessary for the mentalmentor.exe to work properly

The Mental Mentor files are installed into this folder to ensure your app always stays up to date and receives critical updates without extra permissions.
